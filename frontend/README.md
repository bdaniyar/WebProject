# Hotel Booking Frontend

Angular frontend for the Hotel Booking System project.

## Features

- Standalone Angular components
- Routing for `discover`, `login`, and `dashboard`
- JWT authentication with HTTP interceptor
- `HttpClient` service for all API communication
- `FormsModule` + `ngModel` forms
- `@if` and `@for` control flow
- Responsive card-based UI

## Run

```powershell
npm install
npm start
```

The app expects the Django backend to run on `http://127.0.0.1:8000`.

For the full setup flow, see the root [README](../README.md).

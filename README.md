# WebProject

<h1>📖 Project Description: Hotel Booking System</h1>

<p><strong>Hotel Booking System</strong> is a web application that allows users to search, reserve, and manage hotel room bookings online. The platform is built with Angular for the frontend and Django for the backend, ensuring a modern, responsive interface and a reliable server-side API.</p>

<h3>🎯 Project Goals</h3>
<ul>
    <li>Provide a seamless experience for customers to book hotel rooms online</li>
    <li>Enable users to view room availability in real time</li>
    <li>Allow customers to manage their reservations (create, edit, cancel)</li>
    <li>Offer administrators tools to manage hotel rooms, bookings, and users</li>
</ul>

<h3>⚙️ Technologies</h3>
<h5><strong>Frontend:</strong> Angular</h5>
<ul>
    <li>Interactive UI for browsing hotels and rooms</li>
    <li>Calendar view for availability</li>
    <li>User authentication and profile management</li>
    <li>Responsive design for desktop and mobile</li>
    <li></li>
</ul>
<h5><strong>Backend:</strong> Django + Django REST Framework</h5>
<ul>
    <li>RESTful API for handling bookings, users, and hotel data</li>
    <li>Database integration (e.g., PostgreSQL)</li>
    <li>Validation to prevent double-booking</li>
    <li>Admin panel for hotel staff</li>
    <li></li>
</ul>

<h3>🔑 Core Features</h3>
<ul>
    <li>User registration and login</li>
    <li>Hotel and room catalog with details (price, capacity, amenities)</li>
    <li>Calendar view with available dates</li>
    <li>Booking creation, modification, and cancellation</li>
    <li>User dashboard with booking history</li>
    <li>Admin panel for hotel management (add/edit rooms, view reservations)</li>
</ul>
